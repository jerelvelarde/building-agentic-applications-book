- Compile verification is not runtime proof. L1-TOOLS and L3-CHANNELS need fresh browser/Slack captures from the exact lockfile.
- `@copilotkit/channels` `0.1.1` is an early package surface. Recheck source, release status, and supported adapters at publication freeze.
- The Python graph uses `InMemorySaver` for a self-contained excerpt. Production chapters must pair the same pattern with durable storage, authentication, and idempotent external writes.
- Printed excerpts should stay within the named regions; full files remain in the companion so error paths and surrounding controls are not lost.
