# Builder Guide

Use this guide to move from a question to the relevant chapters, executable examples, and external reference projects.

| Path | What you will build or reason about | Chapters |
| --- | --- | --- |
| [Level 1 — Application agents](level-1-application-agents.md) | Web/mobile agentic product with CopilotKit, tools, Generative UI, shared state, and LangGraph | 5–10 |
| [Level 2 — Machine agents](level-2-machine-agents.md) | Governed workspace or CLI agent with skills, policy, containment, and recovery | 11–16 |
| [Level 3 — Organizational agents](level-3-organizational-agents.md) | Channel-native organizational actor with identity, delegation, approvals, and audit | 17–22 |
| [Production engineering](production.md) | Evaluation, security, reliability, and architecture selection across all levels | 23–26 |

Start with [Chapters 1–4](../book-components/sections/01-foundations/) if you need the shared vocabulary: operating surfaces, adaptive execution, the anatomy of an agentic system, and the interface as a control plane.

The companion is intentionally a set of small, tested boundaries rather than one oversized demo. The external projects in [`reference-projects/`](../reference-projects/README.md) show how those boundaries appear in real products.
